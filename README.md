# TrendSense AI 🚀

AI-Powered Business Analytics & Decision Intelligence Platform

## Overview

TrendSense AI is a full-stack SaaS platform that transforms raw business data into actionable insights. Users can upload CSV or Excel datasets, automatically generate KPIs, forecast trends, detect anomalies, analyze customer churn, and interact with their data through an AI-powered Business Copilot.

The platform is designed to help businesses make faster, data-driven decisions without requiring complex analytics tools or coding expertise.

## Installation

### Clone Repository

```bash
git clone https://github.com/kangkan10102001/TrendSense-AI.git
cd TrendSense-AI
```

### Install Dependencies

```bash
npm install
```

or

```bash
bun install
```

### Run Development Server

```bash
npm run dev
```

or

```bash
bun dev
```

### Open Application

```text
http://localhost:3000
```
